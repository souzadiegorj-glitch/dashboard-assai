# Dashboard Assai

Bem-vindo ao Dashboard Assai! Este é um projeto de dashboard desenvolvido para gerenciar e visualizar dados da Assai.

## 🚀 Recursos

- Interface limpa e intuitiva
- Visualização de dados em tempo real
- Responsivo para mobile e desktop
- Gráficos e relatórios personalizáveis

## 📋 Pré-requisitos

- Node.js (versão recomendada: 16+)
- npm ou yarn

## 🛠️ Instalação

```bash
# Clone o repositório
git clone https://github.com/souzadiegorj-glitch/dashboard-assai.git

# Entre no diretório
cd dashboard-assai

# Instale as dependências
npm install
```

## 🏃 Como Executar

```bash
npm start
```

O dashboard estará disponível em `http://localhost:3000`
